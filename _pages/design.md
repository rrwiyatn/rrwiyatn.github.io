Featured Work
======

## Assembly No.1: *a modular chair defined by visible connectors and layered construction*

A fully 3D-printed chair that exposes how it is assembled. Modular components and visible connectors reveal the structural logic of the object while demonstrating how large furniture pieces can be produced using consumer-grade FDM printers.

<p style="margin-bottom: 0.25em;"><strong>Details</strong></p>

<div style="font-size: 0.85em; color: #6b6b6b; margin: 0 0 0.75em; line-height: 1.5;">
Material: PLA<br>
Fabrication: Consumer-grade desktop FDM 3D printing<br>
Construction: Modular components with visible connectors
</div>

This project explores construction as a visible part of the design. Instead of hiding joints and fasteners, the chair highlights them as part of the object’s visual language. Contrasting connectors mark where components meet, making the structure easy to read and emphasizing how the chair is assembled.

The chair is composed of multiple printed elements that interlock to form a rigid frame. Internal cross members support the seat and reinforce the base, while large cutouts reduce visual mass and expose the layered structure inside. These openings allow light to pass through the object and reveal the relationship between the outer shell and the supporting frame.

When viewed through the center of the chair, the internal beams and openings align to reveal the depth of the structure. This perspective exposes the layered construction and reinforces the idea that the internal framework is an integral part of the design rather than something hidden inside.

All components are 3D printed in PLA using consumer-grade FDM printers. Because typical desktop printers have limited build volumes, the chair is divided into smaller printable parts that are later assembled using printed connectors. These connectors function both as structural brackets and as visual elements that clearly express how the object comes together.

Using PLA and widely accessible FDM printers was a deliberate choice. The project investigates how large objects such as furniture can be produced using small, widely available machines by breaking the design into modular components. Rather than concealing the constraints of desktop printing, the chair embraces them, turning assembly and segmentation into defining features of the final form.
